# ML - Previsão de Performance

Projeto desenvolvido para aplicar técnicas de Machine Learning na previsão de desempenho a partir de dados analisados.

## Integrantes

- Luis Fernando França Farias

## Objetivo

Desenvolver um modelo de Machine Learning capaz de analisar dados e realizar previsões de performance, auxiliando na identificação de padrões e tendências.

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Google Colab

## Estrutura do Projeto

- `dataset/` → Base de dados utilizada no treinamento.
- `notebooks/` → Análises e experimentos realizados.
- `modelos/` → Modelos treinados.
- `resultados/` → Métricas e visualizações geradas.

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/luisffarias-dev/ML_Previsao_de_Performace.git


2. Acesse a pasta do projeto:
cd ML_Previsao_de_Performace

3. Instale as dependências:
pip install -r requirements.txt

4. Execute o notebook ou script principal.
